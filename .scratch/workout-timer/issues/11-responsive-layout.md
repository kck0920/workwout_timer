# 11 — 반응형 레이아웃 (모바일/태블릿)

**What to build:** 모바일(< 768px)과 태블릿(>= 768px)에서 최적의 레이아웃을 제공하는 반응형 디자인을 구현한다.

**Blocked by:** 06 — 프리셋 목록 화면, 07 — 프리셋 편집기, 08 — 워크아웃 화면

**Status:** ready-for-agent

- [ ] CSS 미디어 쿼리 브레이크포인트: mobile (< 768px), tablet (>= 768px)
- [ ] 태블릿: 사이드 패널 (프리셋 목록/기록) + 메인 영역 레이아웃
- [ ] 태블릿: 큰 원형 프로그레스 바 + 넓은 터치 영역
- [ ] 모바일: 풀스크린 타이머 레이아웃
- [ ] 터치 컨트롤 적절한 크기 (모바일 최소 44px, 태블릿 최소 48px)
